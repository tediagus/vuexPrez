#### Les modules

Permet de diviser le store en plusieurs petites parties

**Structure du store :**
```js 
store/
    ├── index.js    ---> Main Store file
    └── modules/
        ├── module1.store.js
        ├── module2.store.js
        ├── module3.store.js
        ├── module4.store.js
        ├── module5.store.js
        └── module6.store.js
```
